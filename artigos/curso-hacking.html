Apresentação do curso de hacking

Olá pessoal, tudo tranquilo?

Nesse artigo eu vou apresentar a vocês um curso de hacking que será lançado no youtube.

O foco do curso são as pessoas que querem começar a estudar hacking, mas não sabem o que estudar.

O conteúdo do curso está dividido em 7 aulas.

  • Aula 01 - Apresentando o mundo do hacking
  • Aula 02 - Linux - Kali Linux
  • Aula 03 - Redes de computadores
  • Aula 04 - Desenvolvimento Web
  • Aula 05 - Pentest: Teste de invasão
  • Aula 06 - Python: Criando as próprias ferramentas

Na aula 01 será apresentado um pouco sobre o mundo do hacking de uma perspectiva profissional. Responderei duvidas comuns do tipo: onde um hacker trabalha? onde achar as vagas de emprego?

Na aula 02, vou fazer uma introdução ao Linux, mostrando os principais comandos e conceitos básicos que são importantes conhecer. Vamos usar o Kali Linux, mas os conceitos apresentados servem em qualquer outra distribuição Linux.

Na aula 03, vou falar sobre redes de computadores, o objetivo é que você entenda como a internet funciona, como um dispositivo se comunica com o outro; o que são protocolos; Como eles funcionam e vamos falar sobre a importância de saber redes para o hacking. Vamos usar bastante o wireshark é um pouco do tcpdump para entendermos na prática os conceitos de redes.

Na aula 04, vou ensinar a vocês como as aplicações web são feitas, vamos aprender HTML, CSS, JavaScript, PHP e SQL. Vamos criar uma aplicação web do zero, tanto o front-end quanto o back-end.

Legal, mas qual é a importância de aprender desenvolvimento web?

Então, saber pelo menos o básico sobre programação web é importante para entender como as vulnerabilidades funcionam e por consequência saber como explorar, eu diria até que é o único caminho para não ser só mais um que só sabe usar ferramentas.

Na aula 05, vou mostrar em detalhes a vocês como fazer um Pentest. Um pentest é basicamente um método usado para testar a segurança de um sistema, o objetivo de um pentest é simular um ataque malicioso.

Para conseguirmos fazer isso aprenderemos as técnicas que um cibercriminoso usaria para efetuar a invasão, no nosso caso iremos testar uma aplicação web ou site, como preferir chamar. O objetivo é que você aprenda como explorar manualmente, e de forma automatizada usando ferramentas prontas.

Na aula 06, vou ensinar a programar em Python, o objetivo é ensinar a vocês a criar as próprias ferramentas usando o Python que automatizam a exploração das vulnerabilidades aprendidas na aula 05.

Assim como eu falei no início do artigo, o curso vai começar desde o 0, vou mostra toda a base necessária para você entender como as vulnerabilidades de fato funcionam, vou mostrar também como usar as principais ferramentas, como o Nmap, SQLmap, Wireshark entre várias outras. Mas, nós vamos além, vou ensinar como construir suas próprias ferramentas, criaremos ferramentas capazes de explorar vulnerabilidades como XSS, SQL Injection de forma automatizada.

Bem, por enquanto é isso, espero vocês nos próximos artigos.
