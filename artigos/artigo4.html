O que é OSINT?

OSINT (Open-Source Intelligence), em tradução livre para o português, Inteligência de código aberto é a prática de coletar informações de fontes abertas, ou seja, de fontes que estão publicamente disponíveis, por exemplo, redes sociais, sites, bibliotecas, registro público, a dark web. As técnicas de OSINT podem ser usadas pelo governo, jornalistas investigativos, engenheiros sociais, pentesters, entre outros.

Técnicas de OSINT

O OSINT abrange várias técnicas para fazer a coleta e análise das informações que estão disponíveis publicamente. Segundo a SANS as mais comuns são:

  1. Mecanismos de busca: Mecanismos de busca como Google, Bing e Yahoo são ferramentas valiosas para o OSINT. Usando operadores de pesquisa avançada, os analistas podem filtrar e refinar rapidamente os resultados da pesquisa para encontrar informações relevantes.
  2. Mídias sociais: Plataformas de mídia social como Twitter, Facebook e LinkedIn são fontes valiosas de OSINT. Ao monitorar e analisar a atividade de mídia social, os analistas podem obter informações sobre tendências, sentimentos e ameaças potenciais.
  3. Registros públicos: registros públicos, como documentos judiciais, registros de propriedades e registros de negócios são fontes valiosas de OSINT. Ao acessar esses registros, os analistas podem reunir informações sobre indivíduos, organizações e outras entidades.
  4. Fontes de notícias: Fontes de notícias como jornais, revistas e veículos de notícias on-line são fontes valiosas de OSINT. Ao monitorar e analisar artigos de notícias, os analistas podem obter informações sobre eventos atuais, tendências e ameaças potenciais.
  5. Web Scraping: Web scraping envolve o uso de ferramentas de software para extrair dados de sites. Ao coletar dados de vários sites, os analistas podem coletar grandes quantidades de dados de forma rápida e eficiente.
  6. Ferramentas de análise de dados: Ferramentas de análise de dados como Excel, Tableau e R são valiosas para analisar grandes conjuntos de dados. Usando essas ferramentas, os analistas podem identificar padrões, tendências e relacionamentos nos dados.

Ferramentas

Google

O google é um poderoso mecanismo de busca para os usuários comuns, mas quando se conhece as técnicas de google dorks ele se torna um aliado mais poderoso ainda na hora de coletar informações.

Maltego

Descrição do site do Maltego:

“Maltego é uma ferramenta abrangente para análise de links gráficos que oferece mineração de dados em tempo real e coleta de informações, bem como a representação dessas informações em um gráfico baseado em nós, tornando padrões e conexões de múltiplas ordens entre essas informações facilmente identificáveis.

Com o Maltego, você pode facilmente extrair dados de fontes dispersas, mesclar automaticamente informações correspondentes em um gráfico e mapeá-las visualmente para explorar seu cenário de dados.”

Mitaka

O Mitaka é uma extensão de navegador que auxilia na hora de fazer pesquisas relacionadas ao seu alvo.

SpiderFoot

Descrição do repositório oficial:

“O SpiderFoot é uma ferramenta de automação de inteligência de código aberto (OSINT). Ele se integra a praticamente todas as fontes de dados disponíveis e utiliza uma variedade de métodos para análise de dados, tornando esses dados fáceis de navegar.

O SpiderFoot tem um servidor web embutido para fornecer uma interface limpa e intuitiva baseada na web, mas também pode ser usado completamente através da linha de comando. Ele é escrito em Python 3 e licenciado pelo MIT.”

BuiltWith

O BuiltWith mostra quais tecnologias e ferramentas foram usadas para construir um site.

Intelligence X

O Intelligence X é um mecanismo de pesquisa que busca por vazamentos e informações sobre domínios, endereços IP e endereços de e-mail.

Segundo o próprio site da Intelligence X:

“O Intelligence X diferencia-se de outros motores de busca das seguintes formas:

  1. A pesquisa trabalha com seletores, ou seja, termos de pesquisa específicos, como endereços de e-mail, domínios, URLs, IPs, CIDRs, endereços Bitcoin, hashes IPFS, etc.
  2. Ele faz buscas em locais como a darknet, plataformas de compartilhamento de documentos, dados whois, vazamentos de dados públicos e outros.
  3. Ele mantém um arquivo de dados históricos de resultados, semelhante a como o Wayback Machine de archive.org armazena cópias históricas de sites.”
Recon-ng

Descrição do repositório oficial do GitHub:

“Recon-ng é uma estrutura de reconhecimento completa projetada com o objetivo de fornecer um ambiente poderoso para realizar reconhecimento baseado na web de código aberto de forma rápida e completa.

O Recon-ng tem uma aparência semelhante ao Metasploit Framework, reduzindo a curva de aprendizado para alavancar o framework. No entanto, é bem diferente. O Recon-ng não se destina a competir com os frameworks existentes, uma vez que é projetado exclusivamente para reconhecimento de código aberto baseado na web. Se você quiser explorar, use o Metasploit Framework. Se você deseja fazer engenharia social, use o Social-Engineer Toolkit. Se você quiser realizar reconhecimento, use Recon-ng!”

Shodan

O shodan é um motor de busca usado para pesquisar informações sobre dispositivos conectados à internet, mostrando informações como endereços IP e portas abertas.

Metagoofil

Descrição do repositório oficial do GitHub:

“Metagoofil pesquisa no Google por tipos específicos de arquivos que estão sendo hospedados publicamente em um site e, opcionalmente, os baixa para a sua caixa local. Isso é útil para coleta de Open Source Intelligence, testes de penetração ou determinação de quais arquivos sua organização está vazando para indexadores de pesquisa como o Google.”

theHarvester

Descrição do repositório oficial do GitHub:

“theHarvester é uma ferramenta simples de usar, mas poderosa projetada para ser usada durante a etapa de reconhecimento de uma avaliação de red team ou teste de penetração. Ele executa a coleta de inteligência de código aberto (OSINT) para ajudar a determinar o cenário de ameaças externas de um domínio. A ferramenta reúne nomes, e-mails, IPs, subdomínios e URLs usando vários recursos públicos.”

Seekr

Descrição do repositório do GitHub:

“O Seekr é um kit de ferramentas multiuso para coletar e gerenciar dados OSINT com uma interface web elegante. A nossa vista de ambiente de trabalho permite-lhe ter todas as suas ferramentas OSINT favoritas integradas numa só. O back-end é escrito em Go com BadgerDB como banco de dados e oferece uma ampla gama de recursos para coleta, organização e análise de dados. Se você é um pesquisador, investigador ou apenas alguém que procura reunir informações, o seekr facilita a localização e o gerenciamento dos dados de que você precisa. Experimente e veja como ele pode simplificar seu fluxo de trabalho na OSINT!”

Blackbird

PhoneInfoga

Descrição do repositório oficial do PhoneInfoga no GitHub:

“PhoneInfoga é uma das ferramentas mais avançadas para digitalizar números de telefone internacionais. Ele permite que você primeiro reúna informações básicas, como país, área, operadora e tipo de linha, depois usa várias técnicas para tentar encontrar o provedor VoIP ou identificar o proprietário. Ele funciona com uma coleção de scanners que devem ser configurados para que a ferramenta seja eficaz. O PhoneInfoga não automatiza tudo, está lá apenas para ajudar a investigar os números de telefone.”

Photon

Descrição do perfil oficial do Photon no GitHub:

“Rastreador incrivelmente rápido projetado para OSINT.

O Photon pode extrair os seguintes dados durante o rastreamento:

  • URLs (no escopo e fora do escopo)
  • URLs com parâmetros (example.com/gallery.php?id=2)
  • Intel (e-mails, contas de mídia social, buckets da Amazon etc.)
  • Arquivos (pdf, png, xml etc.)
  • Chaves secretas (chaves auth/API e hashes)
  • Arquivos JavaScript & Pontos de extremidade presentes neles
  • Cadeias de caracteres correspondentes ao padrão regex personalizado
  • Subdomínios & Dados relacionados ao DNS

As informações extraídas são salvas de forma organizada ou podem ser exportadas como json.”

Osintgram

Descrição do repositório oficial do GitHub:

“Osintgram é uma ferramenta OSINT no Instagram para coletar, analisar e executar reconhecimento.

O Osintgram oferece um shell interativo para realizar análises na conta do Instagram de qualquer usuário por seu nickname.

Perguntas Frequentes:

  1. Posso acessar o conteúdo de um perfil privado? Não, você não pode obter informações sobre perfis privados. Você só pode obter informações de um perfil público ou de um perfil que você segue. As ferramentas que dizem ser bem-sucedidas são golpes!
  2. O que é e como posso ignorar o erro challenge_required? O erro significa que o Instagram percebe um comportamento suspeito em seu perfil, então precisa verificar se você é uma pessoa real ou um bot. Para evitar isso, você deve seguir o link sugerido e concluir a operação necessária (inserir um código, confirmar e-mail, etc).”

Telegram Nearby Map

Descrição do repositório oficial da ferramenta:

“O Telegram Nearby Map usa o OpenStreetMap e a biblioteca oficial do Telegram para encontrar a posição dos usuários próximos.

Como funciona?

A cada 25 segundos todos os usuários próximos serão recebidos com TDLib do Telegram. Isso inclui a distância de cada usuário próximo à "minha" localização. Com três distâncias de três pontos diferentes, é possível calcular a posição do usuário próximo. Isso só encontra usuários do Telegram que ativaram o recurso próximo. Por padrão, ele é desativado.”

h8mail

Descrição da ferramenta segundo a sua Wiki:

“O h8mail identifica dados de violação relevantes para a entrada do usuário usando vários serviços de API, bem como pesquisa local multiprocessada. Por padrão, o h8mail usará scylla.sh e hunter.io, o que já é suficiente para identificar violações e encontrar alguma senha.

O h8mail torna-se especialmente poderoso quando usado com chaves de API registradas e dumps de credenciais locais.

Características:

  • Correspondência de padrão de e-mail (reg exp), útil para leitura de outras saídas de ferramentas
  • Passe URLs para localizar e direcionar e-mails diretamente em páginas
  • Padrões frouxos para buscas locais ("john.smith", "evilcorp")
  • Instalação indolor. Requer apenas pip requests
  • Leitura de arquivos em massa para segmentação
  • Saída para arquivo CSV ou JSON
  • Compatível com os scripts torrent "Breach Compilation"
  • Pesquisar texto não criptografado e arquivos .gz compactados localmente usando multiprocessamento. (Compatível com "Collection#1")
  • Receba e-mails relacionados
  • Persiga e-mails relacionados, adicionando-os à pesquisa em andamento
  • Suporta serviços de pesquisa premium para usuários avançados
  • APIs premium de consulta personalizadas. Suporta nome de usuário, hash, ip, domínio e senha e muito mais
  • Reagrupar resultados de violação para todos os destinos e métodos
  • Inclui opção para ocultar senhas para demonstrações
  • Cores deliciosas"
Squatm3gator

Descrição do repositório oficial no GitHub:

“Squatm3gator é uma solução web completa baseada na ferramenta python squatm3, projetada para enumerar domínios disponíveis gerados modificando o nome de domínio original através de diferentes técnicas:

  • Ataques de substituição (Substitution)
  • Ataque de inversão (Flipping)
  • Ataque de homoglifo (Homoglyph) rápido (executar um ataque rápido de homoglifo (homoglyph), mutando apenas uma letra de cada vez)
  • Ataque homoglifo (homoglyph) completo (gera todas as combinações possíveis)

A ferramenta tem como objetivo ajudar testadores de penetração a identificar domínios a serem usados em simulações de ataques de phishing e analistas de segurança para detectar e prevenir ataques de cybersquatting.”

Oblivion

Descrição do repositório oficial:

“O Oblivion é uma ferramenta focada no monitoramento em tempo real de novos vazamentos de dados, notificando se as credenciais do usuário foram vazadas. Também é possível verificar se alguma credencial de usuário foi vazada antes. O Oblivion tem dois modos:

  • Oblivion Client: modo gráfico
  • Oblivion Server: modo com funcionalidades de API

Observação : O cliente de Oblivion e o servidor de Oblivion são independentes.”
